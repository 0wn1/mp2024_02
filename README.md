### About
Some of the [GTA Online: Agents of Sabotage DLC](https://youtu.be/YW9VXUgoG6w) assets converted to FiveM Addons.

### Included:
- SFX Sounds
- Ped Models
- Interiors and Maps
- Vehicles and Animations
